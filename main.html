GIT TOPIC

GIT

GIT adalah sebuah version control system terbuka yang berfungsi sebagai control system untuk menjalankan proyek pengembangan software. Tujuan penggunaan GIT sendiri yaitu untuk mengelola versi source code program dengan menentukan baris serta kode yang akan ditambahkan atau diganti.

Beberapa perintah dasar dalam GIT

1. Git-Init: Digunakan untuk membuat repositori Git baru di direktori lokal.

2. Git Status: Digunakan untuk melihat status dari repositori lokal.

3. Git add: Digunakan untuk menambahkan perubahan atau berkas ke area staging (staging area).

4. Git commit: Digunakan untuk menyimpan perubahan yang telah dilakukan.

5. Git push: Digunakan untuk mengirim perubahan yang ada di repositori lokal ke repositori remote.

6. Git Checkout: Digunakan untuk beralih (checkout) antara branch yang berbeda dalam repositori.

7. Git Merge: Digunakan untuk menggabungkan perubahan dari satu branch ke branch lainnya.

8. Git Clone: Digunakan untuk dapat membuat salinan repository lokal.

9. Git log: Digunakan untuk melihat riwayat komit dalam repositori GIT.

10. Git pull: Digunakan untuk mengambil perubahan terbaru dari repositori remot ke repositori lokal.
